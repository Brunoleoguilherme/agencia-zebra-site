# Agência Zebra Site

Site premium da Agência Zebra em Next.js, pronto para abrir no VS Code e publicar na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra: http://localhost:3000

## Publicar na Vercel

1. Suba a pasta para um repositório no GitHub.
2. Entre na Vercel.
3. Clique em Add New Project.
4. Importe o repositório.
5. Deploy.

## Arquivos principais

- `app/page.tsx` — estrutura das páginas/seções.
- `app/globals.css` — todo o visual premium responsivo.
- `public/assets/logo-zebra.png` — logo da Agência Zebra.
